Description

Text Analysis Skill provides powerful natural language processing capabilities for analyzing, extracting, and understanding text data. Perfect for sentiment analysis, entity recognition, topic modeling, and more.

Key Features

  • Sentiment analysis with confidence scores
  • Named entity recognition (NER) for 30+ entity types
  • Multi-language support including Vietnamese
  • Custom model training and fine-tuning
  • Real-time processing with low latency
  • Batch processing for large datasets
  • REST API and Python SDK
  • Detailed audit logs and analytics

Installation

NPM

npm install @skillhub/text-analysis@2.1.0

Python

pip install skillhub-text-analysis==2.1.0

Docker

docker pull skillhub/text-analysis:2.1.0

Quick Start

Python Example

from skillhub import TextAnalyzer analyzer = TextAnalyzer(model='multilingual') result = analyzer.analyze( text="This product is amazing!", language='en' ) print(result.sentiment) # 'positive' print(result.score) # 0.95

REST API Example

curl -X POST https://api.skillhub.io/v1/analyze \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "text": "This product is amazing!", "language": "en" }'

Use Cases

Customer Feedback Analysis

Automatically analyze customer reviews and feedback to understand sentiment, identify issues, and track customer satisfaction trends.

Content Moderation

Detect and filter inappropriate content in user-generated content with customizable rules and confidence thresholds.

Topic Modeling

Automatically identify and extract topics from large text corpora for content organization and discovery.
